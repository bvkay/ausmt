# Roadmap

AusMT is an active project. This page records the directions it is intended to grow in; it is the
authoritative statement of future direction, and the portal's About page carries the same material
in summary form.

These are directions of travel, not commitments to a schedule. Nothing here describes a shipped
capability — for what exists today, see the relevant Architecture, Data Model and Operations pages.

## Broader validation

Validation of AusMT against a broader range of real-world surveys, metadata records, and
externally authored MTH5 datasets — exercising the framework against data it did not itself
produce, to confirm the metadata and provenance model holds up outside the reference surveys.

## Catalogue expansion

Expansion of the Australian MT catalogue through the addition of historical, legacy, and newly
acquired surveys. Much of the scientific value in Australian MT sits in datasets acquired over the
past four decades; bringing these into a consistent, discoverable framework is a primary goal.

## MTCAT as an open standard

Evolution of the MTCAT JSON schema and discovery API as an open standard for interoperable MT
catalogue discovery and federation. See [Data Model → MTCAT](../data-model/mtcat.md) for the
current shape of the discovery document.

## Federation

Support for federation between independent MT portals while preserving local stewardship,
provenance, and ownership of data holdings — so that a portal can surface another's holdings for
discovery without either party ceding control of its data.
